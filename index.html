// FREQUENTLY ASKED QUESTIONS

$

What shells does bookmark.py support?

bookmark.py supports all major shells including Bash, Zsh, and Fish. It automatically detects your shell during installation and configures the appropriate integration.

~

Where are my bookmarks stored?

Bookmarks are stored in ~/.dir-bookmarks.txt in your home directory. Each bookmark contains the path and name you assigned, making it easy to edit or share if needed.

>

Can I sync bookmarks across machines?

Yes! You can backup your bookmarks using bookmark --backup which creates a timestamped file. Simply copy this file to your new machine and restore with bookmark --restore.

[]

Is bookmark.py fast?

Extremely! bookmark.py is written in Python and optimized for speed. Navigation is instantaneous, and the entire script is only ~50KB with zero dependencies.

#!

Is my data safe?

Absolutely. bookmark.py creates automatic backups before any changes. All operations are validated, and the tool won't let you accidentally delete important directories or create duplicate bookmarks.

*

Can I customize bookmark.py?

Yes! bookmark.py is open source (MIT license). You can modify the script to add features, change the interface, or integrate with other tools. The code is clean and well-documented.
